# react_electron_starter_new          
          
## DEV          
1. First: `npm install`          
2. Webpack: `npm run dev`          
3. Start electron: `npm start`          

## PRODUCT          
1. First: `npm install`                    
2. Webpack: `npm run postinstall`, and change `index.html` to use `bundle.js` built in last step.          
3. Start electron: `npm start`
